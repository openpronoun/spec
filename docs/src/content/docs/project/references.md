- Roden, C. (2024). *Asking users for their pronouns: A guide for UX designers.*
  UX Collective (uxdesign.cc) — best practices for pronoun selectors: optional
  fields, avoiding "Other", allowing multiple and custom pronouns.
- Pew Research Center (2022). Data on Americans' familiarity with gender-neutral
  pronouns — ~20% of Americans know someone using gender-neutral pronouns.
- HL7 International. *Gender Harmony Project — Pronouns Element* (build.fhir.org) —
  defines pronoun as a key demographic element for interoperability and provides
  example pronoun codes (she/her, he/him, they/them).
- PronounDB. *API Documentation* (pronoundb.org) — an existing service that
  standardizes pronoun sets across platforms; illustrates a code-based approach and
  meta entries like "any", "ask", and "avoid".
- University of Washington IT (2023). *Pronoun Data Definition & Integration*
  (it.uw.edu) — handles pronouns as free-form text, advises displaying exactly as
  entered, with a 140-character limit.
- Human Rights Campaign — guide on neopronouns and usage prevalence (hrc.org).
- GLSEN — guidance to avoid the term "preferred pronouns".
